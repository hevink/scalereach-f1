## Recommended Demo Strategy

Pick **6–8 visually distinct niches** for the demo page. Vary both the industry and the ScaleReach features shown:

| Niche | Feature to Highlight |
|---|---|
| Podcast | Captions + intro title |
| Gaming | Split-screen with background |
| Fitness | 9:16 vertical crop + captions |
| Cooking | Smart crop / face detect |
| Motivation | Emoji overlays + captions |
| Tech review | Text overlays |
| Sports | Split-screen + highlights |
| Education | Captions + intro title |

This way prospects from any industry can immediately picture their own content in ScaleReach.
